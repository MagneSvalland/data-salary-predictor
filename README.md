# Data Science Salary Predictor

A machine learning web app that predicts data science salaries based on job title, experience level, location, and company size.

## Tech Stack
- **Python** – data processing and model training
- **Scikit-learn** – regression models
- **Streamlit** – interactive web application
- **Pandas / NumPy** – data analysis

## Features
- Predict salary based on job title, experience, location and company size
- Interactive sliders and dropdowns
- Trained on real-world Kaggle dataset

## Dataset
[Data Science Job Salaries – Kaggle](https://www.kaggle.com/datasets/ruchi798/data-science-job-salaries)
